NAME

zopen-remove − manual page for zopen-remove 0.8.0

SYNOPSIS

zopen remove [OPTION] PACKAGE

DESCRIPTION

zopen−remove − remove installed z/OS Open Tools packages.

PACKAGE is a list of one or more projects to uninstall

OPTIONS

−−help

print this help

−−version

print version

−p, −−purge

remove package, the versioned directory and any cached file

−v

run in verbose mode

EXAMPLES

zopen−remove man−db

Remove the man−db package
