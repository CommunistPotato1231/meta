NAME

zopen-update-cacert − manual page for zopen-update-cacert 0.8.0

DESCRIPTION

zopen-update-cacert:

Update your cacert.pem file to the latest curl CA certificates extracted from Mozilla

Syntax:

zopen−update−cacert [−fhv] [<directory>]

OPTIONS

−f|−−force: update cacert.pem file even if it may be up to date

−h|−−help: print this help

−−version: print version

−v|−−verbose: print verbose messages

Parameters:

<directory>: the directory where the cacert.pem will be updated

The default directory location is: /var/lib/jenkins/workspace/Port−Update−Docs/meta_update
