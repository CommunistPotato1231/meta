NAME

zopen-split-patch − manual page for zopen-split-patch 0.8.0

SYNOPSIS

zopen-split-patch [OPTION] [PATCHFILE]

DESCRIPTION

zopen−split−patch − split a patch file into one patch file for each patch

Patches are written to split−patches in the current directory

OPTIONS

−−help

display this help and exit

−−version

print version

Example:
zopen−split−patch P1.patch

write P1.patch into multiple patches under split−patches
