NAME

zopen-help2man − manual page for zopen-help2man 0.8.0

DESCRIPTION

zopen−help2man will generate man pages from help and version information for zopen and zopen−* tools Syntax: zopen−help2man DIRECTORY

where DIRECTORY is the directory to write the man pages to (the directory needs to already exist)
