NAME

zopen-remove − manual page for zopen-remove 0.8.1

SYNOPSIS

zopen remove [OPTION] [PACKAGE] ...

DESCRIPTION

zopen remove is a utility for z/OS Open Tools to remove an installed package or packages.

OPTIONS

−y, −−yes

automatically answer yes to prompts.

−p, −−purge

remove package, the versioned directory and any cached file

−v, −−verbose

run in verbose mode

−h, −?, −−help

display this help and exit

−−version

print version

EXAMPLES

zopen remove foo bar

interactively remove the foo and bar packages if installed

zopen remove −−yes foo bar

remove the foo and bar packages if installed, without asking for confirmation
