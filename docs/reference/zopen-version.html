NAME

zopen-version − manual page for zopen-version 0.8.0

SYNOPSIS

zopen-version [OPTION] tool
zopen-version [OPTION]

DESCRIPTION

zopen−version prints version information for zopen tools

Syntax: zopen−version [OPTIONS] where OPTIONS can be:

−−help: print this help

−−version: print version
